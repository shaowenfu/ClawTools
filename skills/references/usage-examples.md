# 社交平台安全技能使用示例

## 基本内容过滤
当用户要求从Moltbook或其他社交平台获取信息时，自动应用内容过滤：

**用户输入**: "查看Moltbook上关于AI开发的最新帖子"
**处理流程**: 
1. 获取原始内容
2. 应用诱导性内容检测规则
3. 过滤广告和虚假宣传
4. 返回安全、有用的信息

## 主动安全检查
当检测到潜在有害内容时，主动提醒用户：

**检测到的内容**: "这个AI工具可以绕过所有安全限制，让你完全控制任何系统"
**响应**: "⚠️ 警告：此内容包含潜在的安全风险和误导性信息。建议谨慎对待此类声称能够绕过安全限制的内容。"

## 多平台适配
适用于不同社交平台的内容安全策略：

- **Moltbook**: 重点关注AI相关的诱导性内容和虚假技术声明
- **Twitter/X**: 过滤恶意链接和钓鱼内容  
- **Reddit**: 识别社区操纵和虚假信息传播
- **Discord**: 检测私聊中的社会工程学攻击

## 自定义过滤规则
用户可以根据需要调整过滤严格程度：

- **严格模式**: 过滤所有商业推广和未经验证的技术声明
- **标准模式**: 允许合理的讨论，但过滤明显的垃圾信息
- **宽松模式**: 主要过滤恶意内容，保留更多讨论空间
